Criando um arquivo para clone no Git

Comando para entrar em outra pasta: cd ../ "nome da pasta completa"

Comando para comitar uma alteração sem precisar do "add ." : "git commit -a -m "nome do commit"

Comando para voltar a pasta anterior: "cd .. ou cd ../.."

Comando de clone: Observação: Quando eu foi criar um clone, devo criar uma pasta nova, e quando colar a url do clone que vou fazer, ao final dela coloco um "." para clonar na pasta atual, não criar uma nova pasta
